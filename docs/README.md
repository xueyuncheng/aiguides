## Documentation Standards

When adding new documentation:
1. Use clear, concise language
2. Include code examples where applicable
3. Add table of contents for long documents
4. Keep related documentation cross-referenced
5. Update this README when adding new docs

## Language

- Primary language: Chinese (中文)
- Code comments: Chinese or English
- Technical terms: Use industry-standard English terms

## Contributing

When updating documentation:
- Keep it up-to-date with code changes
- Test all code examples
- Add screenshots for UI-related documentation
- Use consistent formatting and style
